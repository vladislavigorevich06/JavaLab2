## Задачи и реализация

### **1.3 — Имена**

**Описание:**

* Создан класс `Name` с параметрами: фамилия, имя, отчество.
* Метод `toString()` формирует строку вида `"Фамилия Имя Отчество"`.
* Если какой-то параметр отсутствует, он не выводится.

**Примеры объектов:**

```java
Name n1 = new Name(null, "Клеопатра", null);
Name n2 = new Name("Пушкин", "Александр", "Сергеевич");
Name n3 = new Name("Маяковский", "Владимир", null);
```

**Вывод в консоль:**

```
Клеопатра
Пушкин Александр Сергеевич
Маяковский Владимир
```

---

### **2.2 — Человек с именем**

**Описание:**

* Создан класс `Person`, который содержит `Name` и рост.
* Связь имени с человеком реализована через объект `Name`.

**Примеры объектов:**

```java
Person p1 = new Person(n1, 152);
Person p2 = new Person(n2, 167);
Person p3 = new Person(n3, 189);
```

**Вывод:**

```
Имя: Клеопатра, рост: 152
Имя: Пушкин Александр Сергеевич, рост: 167
Имя: Маяковский Владимир, рост: 189
```

---

### **2.3 — Человек с родителем**

**Описание:**

* Добавлен параметр `parent` типа `Person`.
* Метод `fillMissingFromParent()` автоматически заполняет фамилию и отчество:

  * Если нет фамилии — берётся фамилия отца
  * Если нет отчества — формируется от имени отца с суффиксом `"ович"`

**Примеры:**

```java
Person ivan = new Person(new Name("Чудова", "Иван", null), 180);
Person petr = new Person(new Name(null, "Петр", null), 170, ivan);
Person boris = new Person(new Name(null, "Борис", null), 160, petr);
```

**Вывод:**

```
Имя: Чудова Иван, рост: 180
Имя: Чудова Петр Иванович, рост: 170, отец: Иван
Имя: Чудова Борис Петрович, рост: 160, отец: Петр
```

### **3.2 — Ломаная**

* Класс `Polyline` содержит массив `Point`
* Метод `toString()` формирует строку `"Линия [T1,T2,…,TN]"`
* Можно сдвинуть начало ломаной (`shiftStart`)

**Примеры:**

```java
Polyline poly1 = new Polyline(new Point(1,5), new Point(2,8), new Point(5,3));
Polyline poly2 = new Polyline(new Point(1,5), new Point(2,-5), new Point(4,-8), new Point(5,3));
Polyline.shiftStartForPolylines(1,1, poly1, poly2);
```

**Вывод:**

```
Первая ломаная: Линия [{1.0;5.0}, {2.0;8.0}, {5.0;3.0}]
Вторая ломаная: Линия [{1.0;5.0}, {2.0;-5.0}, {4.0;-8.0}, {5.0;3.0}]
После сдвига:
Первая ломаная: Линия [{2.0;6.0}, {2.0;8.0}, {5.0;3.0}]
Вторая ломаная: Линия [{2.0;6.0}, {2.0;-5.0}, {4.0;-8.0}, {5.0;3.0}]
```

---

### **4.9 — Конструкторы ломаной**

* Ломаная может быть создана **без точек** (`Polyline()`)
* Ломаная может быть создана **с набором точек** (`Polyline(Point...)`)

**Пример:**

```java
Polyline emptyPolyline = new Polyline();
Polyline polyWithPoints = new Polyline(new Point(1,5), new Point(2,8));
```

**Вывод:**

```
Пустая ломаная: Линия []
Ломаная с точками: Линия [{1.0;5.0}, {2.0;8.0}]
```

---

### **5.7 — Длина ломаной**

* Метод `length()` считает общую длину ломаной
* Метод `addPoints(Point...)` добавляет новые точки

**Пример:**

```java
Polyline poly3 = new Polyline(new Point(1,5), new Point(2,8), new Point(5,3));
System.out.println(poly3.length());
poly3.addPoints(new Point(5,15), new Point(8,10));
System.out.println(poly3.length());
```

**Вывод:**

```
Длина ломаной: 8.99322955501368
Длина ломаной после добавления точек: 26.824181449858983
```
